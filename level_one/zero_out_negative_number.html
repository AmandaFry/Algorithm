Zero Out Negative Numbers

Assigment description:

  • Given any array x, say [1, 5, 10, -2], create an algorithm that replaces any negative number with the value of 0. When the program is done, x should have no negative values, say [1, 5, 10, 0].
  • Create a for loop to step through each element of the array
  • counter i start with 0 as the array first element is 0 not 1
  • To access the last element of the array is array.length-1
