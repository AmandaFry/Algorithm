Shifting the values in the array (to the left)

Assigment description:

  • Given any array x, say [1, 5, 10, 7, -2], create an algorithm that shifts each number by one to the front. For example, when the program is done, an x of [1, 5, 10, 7, -2] should become [5, 10, 7, -2, 0].
  • Create a for loop to step through each element of the array
  • counter i start with 0 as the array first element is 0 not 1
  • To access the last element of the array is array.length-1
