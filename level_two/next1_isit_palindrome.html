Largest Palindrome

Given a string find the largest palindrome in the string. A palindromic number or numeral palindrome is a number that remains the same when its digits are reversed. Like 16461, for example, it is "symmetrical".

  • One way to do is step thorugh the entire number and flip it first
  • Then compare the original number if it is the same.
  • If yes that is the largest palindrome number if not take one off and compare again until you get to 1 number
  • There must be a way to optimize this I am sure, not
